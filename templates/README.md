# Configuration Templates

These files are **reference templates only**. The webhook reads all configuration from Supabase in real-time.

## Files

- `default-prompt-template.json` - Example chatbot configuration structure
- `clients-template.json` - Example client configuration structure

## Usage

When adding a new client:

1. Go to onmore-admin dashboard
2. Create a new client
3. Use the prompt editor to configure the chatbot
4. Changes take effect immediately (no deployment needed)

## Why Templates?

- Backup reference for configuration structure
- Starting point for new client setups
- Documentation of available fields

**DO NOT edit these files expecting changes to take effect. Use the admin dashboard instead.**
